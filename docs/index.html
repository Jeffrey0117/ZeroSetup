For Developers
你要做的事
只要一次，一行指令。之後使用者完全不需要碰 ZeroSetup。

1. Scan

對你的專案跑一次 init

node zerosetup/init.js your-project/
→

2. Commit

它生成的三個檔案 commit 進去

zerosetup.json
run.bat
stop.bat
→

3. Done

使用者 clone 你的專案，雙擊 run.bat

git clone → run.bat → works
Auto Detection
它怎麼知道你的專案需要什麼
不用手動填設定。掃描你的程式碼就知道了。

Runtime

requirements.txt → Python

Framework

Port

System Tools

npm / pip

Start / Stop

根據 framework 決定啟動和停止指令
