Auto Detection
偵測能力
不需要手動設定任何東西。掃原始碼就知道你需要什麼。

Runtime

偵測 package.json requirements.txt pyproject.toml 判斷 Node.js / Python / both

Framework

識別 PM2 Express Next.js FastAPI Flask Django 自動決定 start/stop 指令

Entry Point

優先 pkg.main 然後掃 server.js app.js main.py 等常見檔名

Port

掃描 .env config.json .listen(PORT) --port flag 多層級偵測

System Dependencies

掃原始碼關鍵字找 cloudflared ffmpeg 自動加入 winget 安裝清單

npm / pip

偵測 pnpm-lock.yaml ecosystem.config.js 決定 npm global 依賴 (pm2, pnpm)

run.bat
四階段啟動
全新電腦只需要 winget（Windows 內建），其他全部自動。

Phase 1 Bootstrap

確認 winget 存在，自動安裝 Node.js 和 Git（如果有 .git）

winget install OpenJS.NodeJS.LTS → REFRESH_PATH → node ready

Phase 2 Read Config

用 node -e 解析 zerosetup.json，轉成 batch 變數

node -e "..." > %TEMP%\zs_vars.bat → call zs_vars.bat

Phase 3 Dependencies

按需安裝 Python、winget 套件、npm globals、npm install、pip install

for %%p in (%ZS_WINGET%) do winget install %%p

Phase 4 Start

執行 pre-start → start command → health check (自動重試 5 次)

call %ZS_START% → Invoke-WebRequest %ZS_HEALTH%

Why It's Cool
四個屌的點

Bootstrap Paradox

run.bat 需要 Node.js 解析 JSON，但 Node.js 可能沒裝。所以 Phase 1 先用純 batch + winget 裝 Node.js，Phase 2 才用 node -e 解析 config。先造出自己的工具，再用工具做事。

winget → node.js → node -e parse JSON → batch vars

Source Code Scanning

不是只看設定檔。掃描原始碼找 cloudflared、ffmpeg 等關鍵字，自動加入 winget 安裝清單。你的 code 用了什麼，它就幫你裝什麼。

source code has "cloudflared" → winget: cloudflare.cloudflared

PATH Registry Refresh

Windows 裝完軟體後 PATH 不即時生效。run.bat 直接從 Registry 撈最新 PATH 覆蓋進來，裝完立刻能用，整個流程不中斷。

reg query HKLM\...\Environment /v Path → set PATH=

One bat, All Projects

run.bat 和 stop.bat 是通用腳本，所有專案共用同一份。Node、Python、PM2、Express、FastAPI 的差異全靠 zerosetup.json 驅動。

same run.bat → read JSON → different behavior per project

Before & After

v1 - 手動模板

1. 複製 run.bat 到專案
2. 複製 zerosetup.config.example.bat
3. 手動設定 APP_NAME=...
4. 手動設定 RUN_MODE=python
5. 手動設定 NEED_PYTHON=1
6. 手動設定 NEED_FFMPEG=1
7. 手動設定 HEALTH_URL=...
8. 每個專案重複以上步驟
